# time2sleep-react
A React + Electron Shutdown Timer

## Install
``` bash
# Clone the repository
$ git clone https://github.com/martinpham97/time2sleep-react

# Go into the repository
$ cd time2sleep-react

# Install dependencies
# Using npm
$ npm install

# Using Yarn
$ yarn install
```

## Run
``` bash
# Using npm
$ npm start

# Using Yarn
$ yarn start
```

## Build
``` bash
# Using npm
$ npm run dist

# Using Yarn
$ yarn dist
```
